Goal of this Project:

1)To create a scalable contact center for our support department capable of handling the following:
a)inbound calls for multiple products
b)Skills based call routing
c)Screen pops
d)OpenCTI
e)CRM Dips to verify support plan status and route call accordingly
f)If call queue is hot, have the customer specify a call back number so the next available agent can contact them back.
g)Create robust chat client that can connect to our Public and Internal KB
h)The chat client to also have a way to switch from chat to call.
i)Be able to report KPIs on all communication channels
j)Be able to log all communication requests as tickets automatically in our support ticket software of choice.
